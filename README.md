A PHP-based health tracking web application built with MVC architecture.

## Features
- Patient login and registration
- BMI calculator
- Health records management
- Appointment booking
- Medication tracking

## How to Run
1. Install XAMPP
2. Clone this repo into htdocs/
3. Start Apache and MySQL
4. Visit http://localhost/healthtracker_phase2/healthtracker/public/
